Schwarmen Racer Turbo

Theme

Schwarmen racer turbo is a 8-bit simulator game which was made for a high school programming project. It has gimmicks like changing seasons, speedometer and highscore system. Presentaion!

Coding

Programming languages we used were JavaScript, CSS and HTML. For more detailed look go to the source link on GitHub.
